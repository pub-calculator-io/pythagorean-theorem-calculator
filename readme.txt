Pythagorean theorem calculator finds the unknown side length of a right triangle. It also calculates angles, area, perimeter, and altitude to hypotenuse.

[https://www.calculator.io/pythagorean-theorem-calculator/](https://www.calculator.io/pythagorean-theorem-calculator/)

== Usage ==

Add the Pythagorean Theorem Calculator shortcode to your page, post or sidebar:

`[ci_pythagorean_theorem_calculator]`

Add the following code to your template where you would like the Pythagorean Theorem Calculator to appear:

`<?php display_ci_pythagorean_theorem_calculator(); ?>`
